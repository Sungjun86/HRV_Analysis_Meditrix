# CSV Reader Android App (Kotlin)

버튼을 누르면 CSV 파일을 선택하고 내용을 읽어 화면에 표시하는 안드로이드 스튜디오 프로젝트입니다.

## 동작 방식
1. `CSV 파일 선택` 버튼 클릭
2. 시스템 파일 선택기에서 CSV 파일 선택
3. 선택된 파일을 읽어서 줄 단위로 파싱 후 화면에 표시

## 실행 방법
1. Android Studio에서 이 폴더 열기
2. Gradle Sync 완료
3. 에뮬레이터/실기기에서 실행

## 참고
- 샘플에서는 단순히 `,` 기준으로 split 하므로, 따옴표 이스케이프 등 복잡한 CSV 규칙은 처리하지 않습니다.
